# AWS SAA-C03 Quiz App

AWS Solution Architect Associate (SAA-C03) 자격증 시험 준비를 위한 퀴즈 애플리케이션입니다.

## 기능

- **724개의 SAA-C03 문제** 수록
- **문제 풀이**: 객관식 문제 선택 및 정답 확인
- **해설 제공**: 각 문제별 상세 해설
- **틀린 문제 관리**: LocalStorage를 활용한 틀린 문제 자동 저장
- **틀린 문제만 풀기**: 복습 모드 지원
- **정답률 통계**: 실시간 정답률 표시
- **문제 네비게이션**: 이전/다음 문제, 특정 번호로 이동

## 사용 방법

### 온라인 사용
GitHub Pages로 배포된 URL에 접속하여 바로 사용할 수 있습니다.

### 로컬 실행
```bash
# 파일을 다운로드한 후
# index.html을 브라우저로 열기
```

## 기술 스택
- Vue.js 3 (CDN)
- Vanilla JavaScript
- LocalStorage (진행 상황 저장)
